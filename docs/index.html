Welcome to Dharma Bums Encoded! This is a project created for a text encoding class where we have encoded the first page of The Dharma Bums by Jack Kerouac using the original manuscript and what was published by Viking Press.

The Original Documents section contains links to the pages with the original manuscript and the published version.

The Final Tables section contains the final product of our work, the tables comparing the two versions of The Dharma Bums.

If you are interested in the code that went into this project visit the "Our Code" section where we show our code and explain how we created this project.
